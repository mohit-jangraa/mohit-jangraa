## 📂 Featured Projects

### 🛍️ [Customer Shopping Behavior Analysis](https://github.com/mohit-jangraa/Customer_behavior_analysis)
> **Python · PostgreSQL · Power BI** &nbsp;|&nbsp; 3,900 rows · 18 columns

Analyzed transactional data from 3,900 purchases across demographics, purchase details, and shopping behavior patterns.

- Cleaned 37 missing values using **category-wise median imputation** in Pandas
- Engineered `age_group` and `purchase_frequency_days` features for better segmentation
- Wrote **10 business SQL queries** in PostgreSQL — revenue by gender (Male: ₹1,57,890 vs Female: ₹75,191), customer segmentation (3,116 Loyal / 701 Returning / 83 New), discount-dependent products
- Built **Power BI dashboard** tracking 3.9K customers, $59.76 avg purchase, 3.75 avg rating with interactive slicers
